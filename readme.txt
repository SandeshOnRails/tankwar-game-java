
# TankWar-Project 

1. Version of Java:     Java 9

2. IDE:    Netbean 8.2

3. Working Directory Path: csc413-03-tankgame-Team01-khanmustufa294

4. Controls:

Player 1:  	        Up Arrow: Move forward
                	Down Arrow: Move backward
                	Right Arrow: Rotate right
                	Left Arrow: Rotate left
                	Enter: Shoot bullets

Player 2:  	        "W": Move forward
                	"S": Move backward
                	"D": Rotate right
                	"A": Rotate left
                	Spacebar: Shoot bullets
